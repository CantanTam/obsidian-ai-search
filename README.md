### [点击查看B站详细设置教程](https://space.bilibili.com/391504171)

<font color="#ff0000ff"><b>［开关弹窗］</b></font>
快速双击 <font color="#de7802">Alt</font> 启动弹窗窗口，弹窗窗口启动后，单击 <font color="#de7802">Alt</font> 、<font color="#de7802">Esc</font> 关闭弹窗窗口；

<font color="#ff0000ff"><b>［查询操作］</b></font>
输入查询内容后 <font color="#de7802">Enter</font> 执行查询操作。未执行查询操作前查询结果输出框会显示上一次查询结果；

<font color="#ff0000ff"><b>［清空输入］</b></font>
输入框长按 <font color="#de7802">空格键</font> 可以清空输入框；

<font color="#ff0000ff"><b>［切换焦点］</b></font>
使用 <font color="#de7802">Tab</font> 键可以在输入框和查询结果输出框之间切换；

<font color="#ff0000ff"><b>［选择文本］</b></font>
在结果输出框当中使用 <font color="#de7802">I</font> 、<font color="#de7802">J</font> 、<font color="#de7802">K</font> , <font color="#de7802">L</font> 可以模拟 <font color="#de7802">↑</font> 、<font color="#de7802">←</font> 、<font color="#de7802">↓</font> 、<font color="#de7802">→</font> 键进行导航；使用 <font color="#de7802">Shift</font> + <font color="#de7802">I</font> 、<font color="#de7802">J</font> 、<font color="#de7802">K</font> , <font color="#de7802">L</font> 可以模拟 <font color="#de7802">Shift</font> + <font color="#de7802">↑</font> 、<font color="#de7802">←</font> 、<font color="#de7802">↓</font> 、<font color="#de7802">→</font> 键进行文本选择；

<font color="#ff0000ff"><b>［发送文本］</b></font>
查询结果输出框的文本处于被选中状态下，按击 <font color="#de7802">空格键</font> 可以将选中文本发送到编辑文档的光标处。如果编辑文档有文本被选中，则发送功能不可用；

<font color="#ff0000ff"><b>［思考模式］</b></font>
某些大模型，例如 <font color="#4f81bd">deepseek-v4-pro</font> 支持 <font color="#4f81bd">thinking</font> 思考模式，点击输入框最右端的小圆圈可以切换思考模式状态，黑色为关闭、绿色为开启。
